# 🕹️ CS2 Server on Azure

This project demonstrates hosting a **Counter-Strike 2 dedicated server** in **Microsoft Azure** using virtual machines and Docker.

## 🎯 Objective
Deploy a CS2 server in Azure to learn VM provisioning, networking, and automation.

## 🧰 Technologies
- Azure Virtual Machines  
- Docker  
- Azure CLI  
- Bash scripting
